# Vencord Plugin Library

A collection of custom and refined Vencord plugins.

## Plugins Included

### 📝 Deletion Logger (Refined)
An enhanced version of the message logger with:
- **Diff View**: See exactly what changed in edited messages.
- **Deletion Timestamps**: Know exactly when a message was deleted.
- **Modern UI**: Clean animations and accessibility fixes.

### 🛡️ Social Monitor
A powerful relationship tracking plugin:
- **Unfriend & Block Alerts**: Get notified if someone removes or blocks you.
- **Mutual Server Leave Notifier**: Know when a friend leaves a shared server.
- **Smart Inference**: Distinguishes between manual removals and external blocks.
- **Themed Toasts**: Color-coded notifications (Red for removals, Yellow for leaves).

### 🤖 AutoQuest Plugin
A professional-grade automation plugin for Discord activities/quests.
